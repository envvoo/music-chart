# 💡 music-chart
메인 페이지 - 멜론 차트 클론코딩


![image](https://github.com/envvoo/music-chart/assets/143536761/b9542bb7-b9df-482f-9296-50a6fb50ed8d)


## 📎 주요 기능
### 차트 갱신 기준 시각
* 시간 단위 갱신
* 시간 강조

### 각 탭의 랭킹 차트
* HTML 파일 분할
* 현재 페이지 강조

### 랭킹 아이템 마우스 이벤트
* 'mouseenter' 일 경우 아이템 구조 전환
  * preset ↔ active
  * 첫 번째 랭킹 아이템의 기본 구조는 active
* 마지막으로 발생한 이벤트 아이템의 상태 고정
  * 다른 아이템의 이벤트 발생까지 active 상태 유지
